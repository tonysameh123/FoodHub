# FoodHub - Full Stack E-Commerce Web Application

FoodHub is an online food ordering web application built using HTML, CSS, JavaScript, Node.js, Express.js, and MongoDB.

## Features

- User registration and login
- JWT authentication
- Browse food menu
- Search and filter meals
- Product details page
- Add to cart
- Checkout and place orders
- Orders stored in MongoDB
- Products stored in MongoDB
- Password hashing using bcrypt

## Tech Stack

### Frontend
- HTML
- CSS
- JavaScript

### Backend
- Node.js
- Express.js
- MongoDB Atlas
- Mongoose
- JWT
- bcryptjs
